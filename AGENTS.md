# AI Agent Instructions for PHP Exercise Repository

## Project overview
- This repository contains a set of standalone PHP exercise scripts.
- Each file is named `ex_01.php` through `ex_06.php`.
- Files define a simple function and then print example output using `echo`.
- There is no framework, dependency manager, or automated test suite in this workspace.

## Key conventions
- Code is written in plain PHP with `<?php ... ?>` tags.
- Function names and output strings are in Portuguese.
- Keep modifications localized to the specific exercise file unless the user explicitly asks for broader refactoring.

## Guidance for AI agents
- Do not assume a build or test command exists; validate by checking the root directory first.
- Preserve the repository structure and filenames.
- When improving code, keep the logic simple and consistent with the existing exercise style.
- If the user asks to add new exercises, follow the existing naming pattern `ex_0N.php`.

## Notes
- There is currently no README or documentation in the workspace.
- Treat this as a learning or practice repository rather than a production application.
